# Features

This page provides detailed information about Carholo's features.

## Interactive 3D Model

The core of Carholo is a fully interactive 3D model of the Porsche 911. Users can:
- Rotate the model by clicking and dragging
- Zoom in and out (if implemented)
- Toggle auto-rotation on/off
- Inspect individual parts by hovering and clicking

## Part Inspection System

### Hover Tooltips
When hovering over car parts, a tooltip appears showing:
- Part name
- Maintenance status (good/warning/damaged)
- Visual status indicator icon
- Color-coded status (green/yellow/red)

### Part Information Panel
Clicking on a part opens a detailed information panel with:
- Part name (large, bold)
- Detailed specifications and information
- Maintenance recommendations
- Animated text reveal effects

### Part Menu
Access all parts via a menu system that shows:
- All available parts
- Status indicators for parts needing attention
- Quick navigation to part information

## Porsche History Article

### "Porsche: The Cult of Precision"
A comprehensive article overlay featuring:
- Long-form content about Porsche history
- Interactive image gallery with parallax effects
- Stock performance chart visualization
- Smooth scroll animations
- Custom scrollbar design

### Image Gallery
- Click images to view in lightbox
- Navigate between images with arrow keys or buttons
- Full-screen viewing with blur effects
- Video support for media content

## Audio Features

### Music Player
- Ambient cyberpunk soundtracks
- Toggle play/pause
- Visual indicator when music is playing
- Smooth transitions between tracks

## User Interface

### Custom Cursor
- Yellow crosshair cursor replaces default pointer
- Color changes based on interaction state:
  - Yellow: Default/neutral
  - Red: Warning or hover over interactive elements
  - Green: Positive status

### UI Elements
- **Fullscreen Button**: Toggle fullscreen mode
- **Rotate Toggle**: Start/stop auto-rotation
- **Info Toggle**: Show/hide part information
- **Book Button**: Open Porsche history article
- **Music Player**: Control background audio

## Visual Effects

- Cyberpunk aesthetic with yellow/gold color scheme
- Glow effects on text and UI elements
- Smooth animations using GSAP
- Backdrop blur effects on overlays
- Image flicker and interlace effects
- Parallax scrolling on full-width images

## Controls

### Mouse
- Click and drag to rotate model
- Hover to see part information
- Click parts to view details
- Click UI buttons for various functions

### Gamepad
- Full gamepad support for navigation
- Controller-friendly interface
